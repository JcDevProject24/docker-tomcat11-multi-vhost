## Proyecto Académico

Este proyecto fue desarrollado como parte del curso de **Despliegue de Aplicaciones Web (DAW)** en el **IES Matemático Puig Adam** durante el año académico 2025-2026.

### Propósito Educativo

Este repositorio tiene fines exclusivamente educativos y de aprendizaje. El código y la configuración aquí presentados demuestran:

- Despliegue de servidores de aplicaciones Jakarta EE
- Configuración de Apache Tomcat 11
- Containerización con Docker
- Implementación de seguridad SSL/TLS
- Gestión de hosts virtuales
